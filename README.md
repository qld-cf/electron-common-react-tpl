# create-react-app + electron

- [x] react 16.12
- [x] electron 9.1.0

#### 运行和打包

```
$ npm i
$ npm start
$ npm run build-react // web构建
$ npm run build-electron // electron构建
$ npm run pack // 当前系统类型打包
$ npm run pack-all // 所有平台类型打包
$ npm run lint // eslint检查
```
